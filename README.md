# Employee-Attrition-and-Performance-Analysis-using-Power-BI

This HR analytics dashboard examines attrition patterns for 1,470 employees across 3 departments using Power BI, Excel, Power Query and DAX, uncovering key workforce risks related to compensation, job level, demographics, performance, and work environment.
